jDateTime

این چیست؟

jDateTime یک لایبری نوشته شده به زبان PHP برای گاه‌شماری جلالی (هجری خورشیدی، تاریخ استاندارد ایران) می‌باشد.
این لایبری براساس لایبری نوشته شده توسط سالار کابلی و کلاس DateTime برای PHP نوشته شده‌است و عملکردی دقیقا همانند تابع DateTime برای تاریخ جلالی دارد.

نصب

نصب این لایبری نیاز به استفاده از composer دارد، بدین منظور پس از نصب کردن composer از کد زیر استفاده نمائید: 
composer require "atofighi/jdatetime":"dev-master"
composer install

استفاده

برای فراخوانی این لایبری از کد زیر استفاده کنید: 
require_once "vendor/autoload.php";
use jDateTime\jDateTime;
حال می‌توانید از این لایبری در کد PHP خود استفاده کنید.

متدها و توابع

توابع این کلاس دقیقا همانند توابع کلاس DateTime است، پس می‌توانید با خواندن راهنمای استفاده از تابع DateTime از این تابع نیز استفاده کنید:
http://php.net/manual/fa/class.datetime.php
